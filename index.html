My Projects

Global Host is a mobile first app that lets travellers connect with guides/hosts in their destination city who share a language and interests, and create itineraries for a day's visit with that host, allowing travellers to experience a city like a local.

Global Host was an eight-day, four-person, final project for Dev Bootcamp. It was built on Rails, utilized the Google Maps JavaScript API, relied on Materialize for visual elements, and was consistently tested using Codeship continuous integration. Check it out on GitHub or try out the App.
